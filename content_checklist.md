# Чек-лист: Работа с контентом (страницы товаров)

## 1. Отображение контента
- [ ] **Главная страница каталога (Inventory):**
  - [ ] Все товары загружаются (по умолчанию 6)
  - [ ] У каждого товара присутствуют: изображение, название, описание, цена, кнопка "Add to cart"
  - [ ] Изображения товаров отображаются корректно (не битые, соответствуют названию)
  - [ ] Цены отображаются в формате $X.XX
  - [ ] Тексты названий и описаний не обрезаны, читаемы

- [ ] **Детальная страница товара (Product Details):**
  - [ ] Крупное изображение товара загружается
  - [ ] Название, описание и цена совпадают с данными в каталоге
  - [ ] Информация о товаре структурирована, нет лишних символов

## 2. Взаимодействие с контентом (добавление в избранное/корзину)
- [ ] **Добавление в корзину из каталога:**
  - [ ] При нажатии "Add to cart" кнопка меняется на "Remove", счётчик в иконке корзины увеличивается
  - [ ] При нажатии "Remove" товар удаляется, счётчик уменьшается
- [ ] **Добавление в корзину из детальной страницы:**
  - [ ] Кнопка "Add to cart" работает аналогично, состояние синхронизируется с каталогом
- [ ] **Множественное добавление:** можно добавить несколько разных товаров, счётчик суммируется

## 3. Функции «поделиться» и социальные кнопки

## 4. Отображение счётчиков (просмотры, лайки)
- [ ] **Счётчик товаров в корзине:** отображается корректно, обновляется при добавлении/удалении
- [ ] **Другие счётчики (лайки, просмотры):** в приложении не реализованы (N/A). Убедиться, что нет ошибочных элементов на этот счёт.
