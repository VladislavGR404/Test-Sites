# Чек-лист тестирования

## 1. Авторизация пользователя (Login Page)

URL: https://www.saucedemo.com/

### Отображение страницы

* [ ] Открывается страница логина
* [ ] Отображается логотип **Swag Labs**
* [ ] Отображается поле **Username**
* [ ] Отображается поле **Password**
* [ ] Отображается кнопка **Login**

### Валидация полей

* [ ] Авторизация невозможна при пустом Username
* [ ] Авторизация невозможна при пустом Password
* [ ] Авторизация невозможна при пустых обоих полях
* [ ] При ошибке отображается сообщение об ошибке
* [ ] Поле Password скрывает вводимые символы

### Позитивный сценарий

Тестовые данные:

```
Username: standard_user
Password: secret_sauce
```

* [ ] Пользователь может войти с валидными данными
* [ ] После входа происходит переход на страницу **Inventory**
* [ ] Отображается список товаров

### Негативные сценарии

* [ ] Вход с неправильным паролем невозможен
* [ ] Вход с неправильным логином невозможен
* [ ] При неверных данных появляется сообщение об ошибке
* [ ] Пользователь `locked_out_user` не может войти в систему
* [ ] При попытке входа отображается соответствующее сообщение
