# 🛒 SauceDemo UI Automation Project

Проект автоматизированного тестирования учебного сайта [SauceDemo](https://www.saucedemo.com/) с использованием **Python**, **Selenium** и **Pytest**.

## 📋 Описание проекта
Этот репозиторий содержит набор приемочных тестов, проверяющих критический функционал интернет-магазина: от авторизации до полного цикла оформления заказа.

### Основные проверки (Test Cases):
* **Авторизация**: Проверка успешного входа под `standard_user`.
* **Обработка ошибок**: Проверка появления сообщения об ошибке при неверном пароле.
* **Сортировка**: Валидация фильтра цен (от низкой к высокой) с проверкой числовых значений.
* **End-to-End (E2E)**: Полный путь покупки: `Добавление в корзину` -> `Checkout` -> `Ввод данных` -> `Финиш`.

## 🛠 Технологический стек
* **Язык**: Python 3.12+
* **Фреймворк**: Pytest
* **Инструмент автоматизации**: Selenium WebDriver
* **Менеджер драйверов**: `webdriver-manager` (автоматическая установка ChromeDriver)

## ⚙️ Установка и запуск

### 1. Клонирование и подготовка
Если вы используете Git, склонируйте репозиторий. Если нет — просто убедитесь, что вы находитесь в папке проекта в терминале.
* ####  Создание виртуального окружения
  ```bash
    python -m venv .venv
    ```
* #### Активация (Windows)
    ```bash
    .venv\Scripts\activate
    ```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
 ```

### 3. Запуск тестов

Вы можете запускать тесты разными способами:
Сами тесты расположены в папке `tests`

| Описание | Команда |
| :--- | :--- |
| **Запустить все тесты** | `pytest` |
| **Подробный вывод (с именами и принтами)** | `pytest -v -s` |
| **Запуск конкретного файла** | `pytest tests/test_checkout.py` |

## 💡 Полезные советы
* **Headless-режим**: Чтобы запустить тесты без открытия окна браузера, перейдите в `conftest.py` и раскомментируйте строку `options.add_argument("--headless")`.
* **Логирование**: В тестах используются паузы `time.sleep()` для визуальной демонстрации. В реальных CI/CD условиях их рекомендуется заменить на явные ожидания `WebDriverWait`.
* **Отчеты**: Если вы хотите сгенерировать HTML-отчет, установите плагин `pip install pytest-html` и запустите команду `pytest --html=report.html`.
