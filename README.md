# Тестирование веб-приложения Swag Labs (SauceDemo)

**Swag Labs**[](https://www.saucedemo.com/) — это демонстрационное веб-приложение в виде интернет-магазина, специально созданное компанией **Sauce Labs** для практики автоматизированного и ручного тестирования.

### Краткое описание сайта
- Тип проекта: веб-приложение (браузерное)
- Основная функциональность: авторизация пользователей, просмотр каталога товаров, сортировка, добавление/удаление товаров в корзину, оформление заказа (checkout), отображение информации о покупателе и финального подтверждения.
- Особенности: несколько тестовых пользователей с разным поведением (стандартный, заблокированный, с глюками производительности и др.), что позволяет отрабатывать негативные сценарии и edge-кейсы.
- Пароль для всех пользователей: `secret_sauce`

Сайт идеально подходит для тренировки навыков автоматизации (Selenium, Playwright, Cypress и др.), написания тест-кейсов, чек-листов, баг-репортов и изучения UI/функционального тестирования.
