
# Чек-лист тестирования


## 1. Работа с профилем пользователя (User Menu / Account)

После авторизации пользователь взаимодействует с аккаунтом через **navigation menu**.

### Открытие меню пользователя

* [ ] Кнопка меню отображается
* [ ] Меню открывается при нажатии
* [ ] В меню отображаются пункты:

  * All Items
  * About
  * Logout
  * Reset App State

### Logout (выход из аккаунта)

* [ ] Пользователь может выйти из системы через кнопку **Logout**
* [ ] После Logout происходит возврат на страницу логина
* [ ] После выхода пользователь не имеет доступа к страницам без повторной авторизации

### Reset App State

* [ ] Кнопка **Reset App State** отображается
* [ ] При нажатии состояние корзины сбрасывается

### Навигация пользователя

* [ ] Переход на **All Items** работает корректно
* [ ] После обновления страницы пользователь остается авторизованным
* [ ] При закрытии сессии требуется повторный вход
